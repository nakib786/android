# Aurora — CRA Mileage Tracker 🍁

**Aurora** is a premium, production-ready mileage tracker app built with Flutter. It's specifically designed for **Canadian users** to maximize tax deductions while remaining fully compliant with **2026 CRA (Canada Revenue Agency)** rules.

## ✨ Features
- **Official 2026 Rates**: Automatically applies $0.73/km (tier 1) and $0.67/km (tier 2).
- **Northern Rates**: Supports higher rates for NT, YT, and NU ($0.77 / $0.71).
- **Split Trip Logic**: Automatically splits trips that cross the 5,000 km threshold.
- **Privacy First**: 100% offline. No accounts, no cloud, no tracking.
- **CRA Reports**: Export PDF logbooks in the official CRA-approved format.
- **Live Tracking**: GPS-based tracking with background support.

## 🚀 Setup Instructions

### 1. Prerequisites
- [Flutter SDK](https://docs.flutter.dev/get-started/install) (v3.0.0 or higher)
- Android Studio / Xcode (for Android/iOS builds)
- Google Maps API Key

### 2. Installation
```bash
# Clone the repository (if applicable)
# git clone <repo-url>

# Install dependencies
flutter pub get

# Generate Isar models (Required)
dart run build_runner build --delete-conflicting-outputs
```

### 3. Google Maps API Key
To enable live tracking and maps, you must add your Google Maps API key:

**Android:**
Edit `android/app/src/main/AndroidManifest.xml`:
```xml
<meta-data android:name="com.google.android.geo.API_KEY"
           android:value="YOUR_KEY_HERE"/>
```

**iOS:**
Edit `ios/Runner/AppDelegate.swift`:
```swift
GMSServices.provideAPIKey("YOUR_KEY_HERE")
```

## 📅 Updating CRA Rates for Next Year
CRA usually announces new rates in mid-January. To update:
1. Open `lib/core/constants/cra_rates.dart`.
2. Update the constants with the new year's rates.
3. Update the `taxYear` constant.

## 📂 CRA Record-Keeping Reminder
> [!IMPORTANT]
> The Canada Revenue Agency (CRA) requires you to keep your mileage logbooks and supporting documents for **6 years** from the end of the tax year they relate to. Aurora's PDF export is designed to fulfill the Motor Vehicle Expense Logbook requirements.
